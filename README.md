# 📊 E-Commerce Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes an E-Commerce Sales dataset using Excel, MySQL, Python, and Power BI. The goal is to identify sales trends, profitable regions, top-selling products, and provide actionable business insights through an interactive dashboard.

---

## 🛠️ Tools Used
- Microsoft Excel (Data Cleaning)
- MySQL (Data Analysis)
- Python (Pandas, Matplotlib)
- Power BI (Interactive Dashboard)

---

## 📂 Project Workflow
1. Data Cleaning using Excel
2. SQL Analysis using MySQL
3. Exploratory Data Analysis (EDA) using Python
4. Interactive Dashboard using Power BI
5. Business Insights & Recommendations

---

## 📈 Dashboard KPIs
- Total Sales
- Total Profit
- Total Orders
- Total Quantity

---

## 📊 Dashboard Visuals
- Sales Trend
- Sales by Category
- Sales by City
- Profit by Region
- Top 10 Products by Sales
- Sales by Payment Mode
- Sales by Sub-Category
- Interactive Filters

---

## 💡 Business Insights
- Home Decor generated the highest sales.
- West region contributed the highest profit.
- Top-selling products generated a significant share of total revenue.
- Top-performing cities contributed most of the sales.
- Business is profitable with positive overall growth.

---

## 🚀 Business Recommendations
- Increase inventory for top-selling products.
- Focus marketing campaigns on low-performing regions.
- Optimize payment methods for better customer experience.
- Improve inventory planning using sales trends.
